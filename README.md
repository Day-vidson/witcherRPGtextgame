# WitcherTextRpg

Tekstowa, turowa gra RPG w C#.

## Zawarte elementy

- klasy i interfejsy z diagramu UML,
- wzorzec projektowy Factory Method,
- podstawowe zasady SOLID,
- walka turowa,
- eksploracja terenów,
- sklep,
- ekwipunek,
- questy,
- system nagród i rankingu.

## Uruchomienie

```bash
dotnet run
```

Projekt używa .NET 8.0.
